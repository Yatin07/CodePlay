# HealthSphere - Doctor-Patient Portal

A comprehensive healthcare platform connecting doctors and patients through secure, real-time communication and medical record management.

## Key Features

### For Patients
- Secure login and profile management
- Schedule and manage appointments
- Video consultations with doctors
- Access to medical records and prescriptions
- Secure messaging with healthcare providers
- Educational health resources

### For Doctors
- Patient management dashboard
- Appointment scheduling and calendar
- Video consultation interface
- Electronic Health Record (EHR) access
- Secure messaging with patients
- Treatment history and notes

## Technical Stack

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Responsive design for all devices

### Backend
- Node.js (Server.mjs)
- Firebase Authentication & Realtime Database
- RESTful API architecture

## Project Structure

### Core Files
- `index.html` - Main application entry point
- `Server.mjs` - Backend server implementation
- `login.html` - Authentication interface
- `dashboard.html` - Main dashboard for doctors/patients

### Key Components
- `ehr.html` & `ehr2.html` - Electronic Health Records management
- `sched.html` - Appointment scheduling
- `vdo.html` - Video consultation interface
- `messege box.html` - Secure messaging system
- `edures.html` - Educational resources

## Setup & Installation

1. **Prerequisites**
   - Node.js (v14+)
   - npm (v6+)
   - Firebase account and project

2. **Installation**
   ```bash
   git clone https://github.com/Yatin07/CodePlay.git
   cd CodePlay
   npm install
   ```

3. **Running Locally**
   ```bash
   node Server.mjs
   ```
   Open `http://localhost:3000` in your browser
